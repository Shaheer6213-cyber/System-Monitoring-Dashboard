# 💻 System Monitoring Dashboard

## 📌 Project Description
This project is a Windows-based system monitoring dashboard developed in C++. It monitors system performance including CPU usage, RAM usage, and Disk usage in real-time.

## 🎯 Features
- Real-time CPU monitoring
- RAM usage tracking
- Disk usage monitoring
- Alert system (beep/text notification)
- Logging system

## 🛠️ Technologies Used
- C++
- Windows API
- GDI+ (for GUI)
- SAPI (for voice alerts)

## 📂 Project Structure
- src/ → Source code files
- docs/ → Project proposal and documentation

## ⚙️ How to Run
1. Open project in Visual Studio
2. Build the project
3. Run the executable file
